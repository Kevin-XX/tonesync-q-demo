**方案**：

│                                     │

│                                     │
│  ⚡ 快捷功能            [查看全部 ↓] │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐      │
│  │聊天│ │群聊│ │事件│ │创作│      │
│  │助手│ │管家│ │    │ │    │      │
│  └────┘ └────┘ └────┘ └────┘      │
│                                     │
│  🎯 小Q养成进度                      │

1. 只显示4个最常用功能，其他折叠
