# 🤖 同频小Q - ToneSync Q

> 腾讯PCG校园AI产品创意大赛决赛路演 Demo

一款基于 QQ 生态的智能社交助手，以 AI 为翼，让社交更有温度、更具想象力。

## ✨ 核心功能

| 功能模块 | 描述 |
|---------|------|
| **智能聊天助手** | 智能回复建议、文案润色、话题推荐、表情推荐 |
| **群聊管家** | 群聊智能摘要、关键信息提取、投票管理、群事件播报 |
| **社交事件触发** | 自动感知生日/纪念日，生成个性化互动建议 |
| **小Q养成系统** | 人格养成、能力成长、形象定制、成就系统 |

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 生产构建

```bash
npm run build
```

## 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **样式**: Tailwind CSS 3
- **动画**: Framer Motion
- **图标**: Lucide React

## 🎨 设计亮点

- 🔵 **QQ蓝主色调** - 统一的品牌视觉
- 🔮 **毛玻璃效果** - 现代感 UI 设计
- ✨ **微交互动画** - 流畅的用户体验
- 📱 **移动端优先** - 响应式设计

## 🎯 产品理念

同频小Q致力于成为用户的智能社交伙伴，通过 AI 技术：

1. **降低社交门槛** - 智能回复建议，轻松开启对话
2. **提升社交质量** - 精准把握语境，让交流更有温度
3. **丰富社交体验** - 个性化互动，让每一次交流都与众不同
